🌍 AQI Data Analysis & Visualization Project
📌 Project Overview

This project focuses on analyzing Air Quality Index (AQI) data by building a complete data pipeline from raw data ingestion to visualization. The project integrates SQL, Python, Excel, and Power BI to clean, process, analyze, and visualize air pollution data for meaningful insights.

The workflow ensures data accuracy by maintaining both raw and cleaned datasets, enabling reliable analysis and comparison.

🎯 Objectives

Store and manage raw AQI data securely

Clean and standardize data for accurate analysis

Perform advanced SQL-based analytics

Generate insights using Python visualizations

Build interactive dashboards using cleaned data in Excel & Power BI

🛠️ Tools & Technologies Used

MySQL (SQL) – Data storage, querying, and analytics

Python (Jupyter Notebook) – Data extraction, cleaning, and visualization

Excel – Dashboard creation using cleaned data

🔄 Project Workflow

Data Ingestion: Imported raw AQI data into MySQL and validated it.

Data Extraction: Retrieved data from MySQL using Python.

Data Cleaning: Handled missing values, standardized data, and created new features.

Data Storage: Stored cleaned data back into MySQL.

Data Analysis: Performed advanced SQL queries for insights.

Visualization: Created charts using Python.

Dashboard: Built interactive dashboards in Excel and Power BI using cleaned data.

📊 Key Insights

Identified highly polluted cities

Observed seasonal variations in AQI

Highlighted major contributing pollutants

Improved accuracy through data cleaning

Clear difference between raw and cleaned data analysis


Power BI – Interactive and advanced dashboards
